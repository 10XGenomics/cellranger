# Building Cell Ranger
## Build dependencies
- Python 2.7
- rust 1.15.1
- gcc 4.8.3
- go 1.5.1

## Build command
`make`

# Running Cell Ranger
## Runtime dependencies
- Binary dependencies can be found in the Ranger package (https://support.10xgenomics.com/developers/software/downloads/latest)
